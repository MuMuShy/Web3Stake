# 實作ERC-20發幣&彈性質押
專案發布在 Goerli測試鏈


網站: https://mumu.tw/ReactProject/Web3Stake/

## 功能

https://app.uniswap.org/#/swap


將metamask切換成Goerli測試網 點選token並輸入MUMU代幣合約地址: 0xF9f4Bb1aDCc0527FCa576532ea784EE90c1D7900
